# 概要   
1. ランダムに株銘柄を選択して、その情報を表示します。  
2. 「株価チャート（移動平均線）」と「価格帯別の出来高」を計算して、2つのグラフを作成します。  
3. 時系列データ分析のPythonライブラリ「prophet」を用いた株価予測をします。  
4. ニューラルネットワークのPythonライブラリ「keras」を用いて、RNN(Recurrent Neural Network)による株価予測をします。  

# 使い方
次のURLをクリックすると実行されます。  
https://checkstockpricechart-kqdzrm9txwaqeubt4b3uls.streamlit.app/  

# 備考
株銘柄はランダムに選択されます。そのため、再読み込みする度に、表示される銘柄は異なります。
